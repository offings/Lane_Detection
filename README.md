## POSCO AI BIGDATA 아카데미 17기(AI 프로젝트)
## 개발 추진 배경
* 시각안내견을 대신할 4족 보행 로봇에 탑재할 기능 개발 필요
* 실시간으로 도로 상황을 파악하고 시각장애인에게 정확한 위치와 장애물 안내 필요

## 프로젝트 개발 목표
* OpenCV를 활용하여 차선 검출과 점자 블록 인식을 진행하여 시각장애인에게 안전한 보행 경로를 안내하고자 한다.
